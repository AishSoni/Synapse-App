Synapse: Desktop Dashboard UI Design Document (Mind Map & Spaces)Goal: To create an aesthetically pleasing, ultra-fast, and frictionless user interface optimized specifically for an Electron desktop environment, focusing on the visualization of semantic relationships and user-defined organization (Spaces).ElementSpecificationRationaleThemeDark Mode (bg-gray-900/bg-gray-800).Best for prolonged desktop use, aligning with most modern creative/research applications.Color PalettePrimary Accent: Neon Blue/Violet (indigo-500/#6366f1). Secondary: Light Gray/White.High contrast, digital aesthetic that works well in a dark desktop setting.GeometryRounded Corners (rounded-xl or rounded-2xl). Minimal Borders.Provides a clean, modern, "floating card" feel, characteristic of native desktop UIs.PerformanceMinimal CSS animations (outside of the chat). Focus on instant visual response.Essential for meeting the "speed and reliability" core expectation of a desktop app.Core Component Focus: The Intelligent Mind MapThe Mind Map visualization is the centerpiece, moving beyond simple storage to display context and relationships.ElementVisual StyleFunction & InteractionNode (Snapshot)Rounded card or circle, colored based on the containing Space. Text label shows content type (e.g., 'PDF Snippet', 'Quote', 'Product Card').Represents a single piece of captured content. Click to open a detail panel/reader view.Semantic Link (AI-Driven)Thin, glowing lines (e.g., text-indigo-400 with subtle shadow) connecting nodes.Automatically generated by the AI backend based on content relevance (e.g., matching keywords, similar topics, or shared source context).Space (User/AI Grouping)A visual cluster of connected nodes, often contained within a subtly shaded, named boundary on the map (like a cloud or box).User-Defined Curation: A living, named collection (e.g., "AI Agents Research"). New nodes can be manually dragged into a Space or auto-included by AI based on the Space's defining query.Interaction: Grouping Nodes into SpacesSelection: Users select multiple nodes on the canvas (simulated via an action button in this prototype).Grouping Action: The user clicks a dedicated "Group Selected Nodes" button.Naming: A modal (simulated by a browser message) prompts the user to name the new collection, creating a persistent Space on the map and in the sidebar navigation.UI Component SummaryFixed Side Panel (20rem): Contains Chats/Spaces (including a list of user-defined Spaces), Settings, API & Auth, and the new Setup Guide button.Mind Map Canvas: The primary interactive visualization area, showing nodes and semantic links.Floating Chat Window (Bottom Center): The fast, intelligent search interface, expanding up and sideways to 50% of the viewport.
